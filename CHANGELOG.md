## [1.0.0] — 2026-07-21

الإصدار الأول 🎉

### ✦ المحتوى
- بوستات بنص وصور (لحد ٤) وفيديو، مع ضغط الصور في المتصفح قبل الرفع
- استطلاعات رأي بمدة محددة ونتائج لحظية ومنع التصويت المزدوج
- اقتباس البوستات (Quote) والريبوست
- تعليقات بمستوى ردود واحد، وكل تعليق بيتعمله لايك
- استخراج تلقائي للهاشتاجات **بالعربي والإنجليزي** والمنشنات `@`
- محفوظات خاصة
- عدادات مشاهدة بتتسجّل لما البوست يظهر فعلًا على الشاشة

### ✦ الستوريز
- ستوريز ٢٤ ساعة: صورة، فيديو، أو نص ملوّن بـ ٦ خلفيات
- عارض بشرايط تقدم، إيقاف بالضغط المطول، وتنقل بالكيبورد
- قايمة مين شاف الستوري

### ✦ التواصل
- رسايل خاصة لحظية
- مؤشر "بيكتب دلوقتي…" (Broadcast)
- علامات القراءة ✓✓
- إرسال صور في الشات
- حالة التواجد (أونلاين 🟢) عبر Presence
- إشعارات لحظية لـ ٨ أنواع تفاعل، بتوست وصوت وبادج

### ✦ الاكتشاف
- فيد **"ليك"** بخوارزمية تفاعل + حداثة في قاعدة البيانات
- فيد **"بتتابعهم"** بيدمج البوستات والريبوستات مرتبة زمنيًا
- ترندات آخر ٤٨ ساعة
- بحث شامل: ناس، بوستات، هاشتاجات
- اقتراحات متابعة

### ✦ الحساب
- تسجيل ودخول بالإيميل، استرجاع الباسورد، إعادة إرسال التفعيل
- فحص توفر اسم المستخدم لحظيًا
- بروفايل بصورة وكوفر وبايو، بتابات (بوستات / ميديا / لايكات / محفوظات)
- ستريك النشر اليومي 🔥
- **أول مستخدم بياخد شارة التوثيق تلقائيًا** ✦

### ✦ التصميم
- نظام تصميم كامل مستوحى من بوسترات البوب-آرت المصرية الريترو
- وضع ليلي بسماء نجوم
- أصوات ريترو مولّدة بـ Web Audio (مفيش ملفات صوت)
- ريسبونسڤ كامل بنڤ سفلي وزرار عائم على الموبايل
- PWA قابلة للتثبيت مع Service Worker
- احترام `prefers-reduced-motion`

### ✦ التقني
- ١٦ جدول مع Triggers لكل العدادات والإشعارات
- ٨ دوال RPC
- RLS متفعّل على كل الجداول، والدوال الداخلية محجوبة عن الـ API
- Storage buckets للميديا والأفاتارات بصلاحيات مقيدة لكل مستخدم
- تجميع دفعي للاستعلامات: ٤ استعلامات لأي فيد مهما كان عدد البوستات
- كاش للبروفايلات وطابور للمشاهدات
- **صفر build step، صفر تبعيات وقت التشغيل** (غير Supabase SDK من CDN)
