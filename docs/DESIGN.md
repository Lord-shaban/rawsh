# نظام التصميم 🎨

روش مش موقع فيه ألوان حلوة — ده **بوستر مصري ريترو اتحوّل لواجهة**.

---

## المرجع البصري

الهوية مستوحاة من بوسترات البوب-آرت المصرية (Y2K / التسعينات):

- ورق مطبوع بخامة **جرين** (حبيبات) واضحة
- خط عربي **تخين جدًا** بظل ثلاثي الأبعاد متدرج
- ستيكرز بحدود سودا سميكة وظل قاسي مزاح
- **نجوم وسبارك** ✦ خلف العناصر المهمة
- شطرنج، هالفتون (نقط)، وشرايط LED
- ديالوجات ويندوز قديمة بشريط عنوان متدرج
- ألوان **صريخة** فوق ورق كريمي هادي

القاعدة الذهبية: **لو الحاجة تقدر تتزوّق شوية، زوّقها.** بس خليها مقروءة.

## المكوّنات الأساسية

### الستيكر — أساس كل حاجة

```css
.sticker {
  background: var(--card);
  border: 2.5px solid var(--line);
  border-radius: 18px;
  box-shadow: -5px 5px 0 var(--sh);   /* ظل قاسي، مش ضبابي */
}
```

**الظل بينزاح لليسار** (`-5px`) مش لليمين — لأن الموقع RTL، والضوء جاي من ناحية اليمين.

### الأزرار

الزرار بيتحرك زي ما تكون بتضغط على ستيكر حقيقي:

```css
.btn:hover  { transform: translate(1px, -1px);  box-shadow: -6px 6px 0; }  /* بيرتفع */
.btn:active { transform: translate(-4px, 4px);  box-shadow: 0 0 0;      }  /* بينضغط والظل بيختفي */
```

الأصناف: `.btn-or` `.btn-pk` `.btn-bl` `.btn-yl` `.btn-dark` `.btn-danger` `.btn-ghost`
الأحجام: `.btn-sm` `.btn-lg` `.btn-icon`

### النص ثلاثي الأبعاد

```css
.t3d {
  color: var(--yl);
  -webkit-text-stroke: 1px var(--ink-const);
  text-shadow:
    -1px 1px 0 var(--ink-const),
    -2px 2px 0 var(--or), -3px 3px 0 var(--or),
    -4px 4px 0 var(--or), -5px 5px 0 var(--or2),
    -6px 6px 0 var(--ink-const),
    -10px 12px 18px rgba(0,0,0,.35);
}
```

طبقات ظل متتالية بتبني العمق، وآخر طبقة ضبابية بتدي إحساس إن الحروف طايرة فوق الورق. النسخ: `.t3d` (برتقالي) · `.t3d-pk` (بينك) · `.t3d-bl` (أزرق).

### نافذة ويندوز

`.win` مع `.win-bar` — شريط عنوان متدرج بـ ٣ دواير ملونة وزرار ✕ أحمر. كل المودالات بتستخدمها.

### الخامات

| الكلاس | الشكل |
| --- | --- |
| `.checker` | شطرنج (كوفر البروفايل الافتراضي) |
| `.halftone` | نقط مطبعية |
| `.scanlines` | خطوط شاشة قديمة |
| `.grain` | طبقة الحبيبات الثابتة فوق الصفحة كلها |

الجرين نفسه SVG `feTurbulence` مدمج كـ data URI — **مفيش أي ملف صورة في المشروع كله**.

## الأصوات

مفيش ملفات صوت خالص — كلها مولّدة بـ **Web Audio API** في [`js/lib.js`](../js/lib.js):

| الصوت | الشكل |
| --- | --- |
| `sfx.like()` | نغمتين مربعة صاعدة |
| `sfx.pop()` | مثلثية قصيرة |
| `sfx.send()` | جيبية بتزحف من 320 لـ 780 هرتز |
| `sfx.msg()` | رنة استقبال |
| `sfx.notif()` | نغمتين تنبيه |
| `sfx.err()` | نغمة منشارية منخفضة |

كلها بجين خفيف (0.035–0.05) عشان ماتزهقش، وممكن تتقفل من الإعدادات.

## قواعد لما تضيف حاجة جديدة

1. **استخدم المتغيرات** — مفيش لون مكتوب بإيدك في الكود
2. **الحدود 2.5px والظل `-Npx Npx 0`** — مفيش ظل ضبابي
3. **جرّبها في الوضعين** — الليلي مش مجرد عكس ألوان
4. **الألوان الصريخة تاخد `--ink-const`** فوقها مش `--ink`
5. **الأنيميشن يحترم `prefers-reduced-motion`**
6. **زوّدها إيموجي** — الإيموجي جزء من الهوية مش زيادة
